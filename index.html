상품

장기 보험이란?

사고 위험 보장에 거축 기능을 추가할 수 있는 보험이며 보험 기간이 3년 이상인 보험을 말합니다.

# 통합보험 # 상해·질병 # 보장성 # 저축성 # 화재·재물보험 # 연금

자동차 보험이란?

자동차 소유, 사용, 관리하는 동안 발생하는 법률상 손해배상을 위한 보험입니다.
대인배상, 대물배상, 자기신체사고, 자기차량손해, 무보험 자동차에 의한 상해 5가지입니다.

# 대인배상 # 대물배상 # 자기신체사고 # 자기차량손해 # 무보험 자동차에 의한 상해

실손 의료보험이란?

질병, 상해로 입원하거나 통원 치료를 하는 경우 피보험자가 실제 부담한 의료비를 보험금으로 지급하는 보험입니다.
요양급여, 의료급여 중 본인부담액, 비급여 내 자기부담금을 차감한 후 받을 수 있습니다.

# 4세대 실손 # 유병자 실손 # 1·2·3세대 실손 계약전환 # 노후 실손

32개 보험사와 제휴하여, 생명 및 손해보험사
최적의 상품 서비스를 제공하고 있습니다.

계약

보험 분석이란?

나의 보험 가입정보를 My Data와 연동하여 보장 및 가입금액의 적정성을 상담 제공해드립니다.

# 마이데이터 # 보장분석 리포트

비갱신형/갱신형이란?

비갱신은 만기까지 동일한 보험료를 납부하며, 갱신형은 납입기간동안 달라질 수 있습니다.
갱신형 초기 보험료가 저렴하나 의료수가 변동과 갱신시점 연령을 적용하여 인상될 수 있습니다.

# 비갱신형 # 갱신형

계약 관리란?

국내 32개 보험사와 제휴하여, 최적의 상품 제공은 물론 계약관리를 One-Stop으로 도와 드립니다.
전국 지점에서 대면, 비대면을 통하여 서비스를 제공해드립니다.

# 생명보험 # 손해보험

납입 및 주요 보장에 대한 상담을
One-Stop 계약 관리하세요.

보험금

보험금 안내 서비스

복잡한 보험금 청구 절차는 보험 전문가와 사전 상담을 통하여 보상 여부 등을 체크해보세요.

# 진단비 # 수술비 # 실손의료비 # 배상책임

예상보험금 안내

각종 사고와 질병으로 발생할 예상 보험금을 확인해보세요.
질병, 수술을 경험하였거나 대비할 경우, 미리 확인해보시기 바랍니다.

# 내 보험금 찾기 # 예상 보험금 # 미청구 금액

청구서류 안내 서비스

보험 사고의 원인에 따라 각 보험사별 청구 서류의 요청이 상이할 수 있습니다.
가입한 회사에 맞는 청구 서류를 안내 드립니다.

# 약관 # 청구 절차 # 제출서류
